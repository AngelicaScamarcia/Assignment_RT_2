case_2 Module

Service:

/kb_input

This node implements the lauching of:

case 2. the user can drive the robot as he wants without any constraints case 3. the user drives the robot with the collision control, that is able to avoid the user to hit obstacles.

/remap_cmd_vel to send the velocity in different cases based on the scanning /scan to get the laser scanning of the map

Publishes to:

This node implements the driving of the robot with the obstacle avoidance.
